# rsofun BiomeE (P-model) output data

Example output dataset from a BiomeE-model run using divers
[`biomee_p_model_drivers`](https://fabern.github.io/rsofun/reference/biomee_p_model_drivers.md)
See
[`runread_biomee_f`](https://fabern.github.io/rsofun/reference/runread_biomee_f.md)
and
[`run_biomee_f_bysite`](https://fabern.github.io/rsofun/reference/run_biomee_f_bysite.md)
for a detailed description of the outputs.

## Usage

``` r
biomee_p_model_output
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with 1
rows and 2 columns.
