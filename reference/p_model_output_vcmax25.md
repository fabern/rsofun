# rsofun P-model output data (using vcmax25 drivers)

Example output dataset from a p-model run using
[`p_model_drivers_vcmax25`](https://fabern.github.io/rsofun/reference/p_model_drivers_vcmax25.md)
See
[`run_pmodel_f_bysite`](https://fabern.github.io/rsofun/reference/run_pmodel_f_bysite.md)
for a detailed description of the outputs.

## Usage

``` r
p_model_output_vcmax25
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with 4
rows and 3 columns.
